Fluent design system

A simple way to make your application written in WPF keep up with modern design trends. Library changes the base elements like Page, ToggleButton or List, and also includes additional controls like Navigation, NumberBox, Dialog or Snackbar.

About

Users are the principal model implemented here. It is needed to perform any other task in this API.

Getting started

Users can be created by issuing a POST request to the /users endpoint. An user is required to have a name. Users with same name are allowed to coexist.

Attributes

  • name string

  • Name of the user being created
